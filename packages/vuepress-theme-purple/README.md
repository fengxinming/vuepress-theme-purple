## Usage

```
yarn add vuepress-theme-purple -D
# OR npm install vuepress-theme-purple -D
```

**edit `.vuepress/config.js`**
```js
// .vuepress/config.js
module.exports = {
  theme: 'purple',
  themeConfig: {
    // Please head documentation to see the available options.
  }
}
```

Same with:

```js
// .vuepress/config.js
module.exports = {
  theme: 'vuepress-theme-purple',
  themeConfig: {
    // Please head documentation to see the available options.
  }
}
```
